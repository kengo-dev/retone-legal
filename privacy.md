# ReTone プライバシーポリシー

最終更新日: 2026年4月8日

本アプリ（ReTone / リトーン）は、お客様のプライバシーを尊重し、個人情報の保護に努めます。本ポリシーでは、本アプリが取得する情報とその利用方法について説明します。

---

## 第1条（収集する情報）

本アプリは、以下の情報を収集・利用します。

### 1.1 お客様から直接収集する情報（アプリの操作を通じて）

1. **返信支援のための入力**
   * 受信メッセージ等のテキスト（手入力）
   * スクリーンショット等の画像（OCR によりテキスト化する目的でアップロードされる場合）
   * 相手との関係、返信の目的、トーン、長さ等の選択・設定

2. **生成結果・履歴に関する情報**
   * 返信候補、リスクに関する分析結果、「次の一手」等の提案内容
   * 履歴として保存される生成結果に関連するメタデータ（保存日時等）

3. **アプリ設定**
   * 既定の関係・トーン・長さ、OCR の利用可否、履歴保存の可否、その他アプリ内設定

4. **サブスクリプションに関する情報**
   * プラン種別、有効性の確認に必要な情報（**決済そのものは Apple の App Store が処理**し、本アプリがカード番号等を取得することはありません）

### 1.2 自動的に収集する情報

* **端末識別子**: 利用回数の管理、履歴・設定・課金状態の照会等のため、端末に紐づく識別子を Keychain 等に保存し、API リクエストに含める場合があります（メールアドレスや氏名によるログインは行いません）。

* **デバイス情報**: OS 情報、アプリバージョン、プラットフォーム情報等

* **利用・技術情報**: エラー情報、パフォーマンスに関する情報、API 利用に伴うログ

* **ネットワーク情報**: IP アドレス、接続状態（オンラインであることの確認等）

## 第2条（情報の利用目的）

収集した情報は、以下の目的で利用します。

* **サービス提供**: 返信候補の生成、リスク分析、次の一手の提案、OCR、履歴の保存・表示、利用回数制限の適用
* **課金・契約**: サブスクリプション状態の確認、App Store との整合性確認
* **改善・開発**: サービスの品質向上、不具合修正、利用実態に基づく改善（個人を特定しない形での集計・分析を含む場合があります）
* **サポート**: お客様からのお問い合わせへの対応
* **セキュリティ**: 不正利用の防止、濫用検知、サービスの安定運用
* **法的義務**: 法令に基づく対応

### 2.1 第三者AI/OCRサービス送信前の同意

本アプリは、第三者サービス（Amazon Bedrock / Google Cloud Vision API）へお客様データを送信する前に、アプリ内の同意画面で以下を明示し、同意を取得します。

* 送信されるデータ項目
* 送信先サービス名
* 利用目的

お客様は、同意後であっても設定画面から同意状態を変更できます。不同意の場合、当該外部連携を必要とする機能（返信候補生成、リスク分析、OCR）は利用できません。

## 第3条（情報の保存・管理）

### 3.1 データの保存場所

* **端末内保存**:
  * 設定値等: UserDefaults 等、OS が提供する仕組み
  * 端末識別子: Keychain 等、OS が提供する仕組み

* **クラウド保存（開発者が利用する AWS 等）**:
  * 履歴、設定の同期・保存、利用回数管理、課金状態に関する記録等は、**Amazon Web Services（AWS）** 上のサービス（例: Amazon DynamoDB、AWS Lambda 等）を通じて処理・保存される場合があります。
  * AWS の主たるリージョンは **アジアパシフィック（東京）ap-northeast-1** を想定していますが、サービス構成や AWS の冗長化・バックアップ等により、データが他リージョンで処理または複製される場合があります。

### 3.2 データの暗号化・通信

* 本アプリとサーバーとの通信は、**HTTPS（TLS）** により暗号化されます。
* クラウド上のデータは、AWS 等が提供する暗号化・権限管理の仕組みにより保護されます。

### 3.3 データの保存期間

* **履歴・設定・利用管理に必要なデータ**: お客様がアプリ内で削除するまで、または開発者がサービス運営上合理的に必要と判断する期間。一定期間経過後に自動削除（TTL）される設計となる場合があります。
* **課金・取引の記録**: 税法、会計、紛争対応等に必要な期間、または App Store / Apple の方針に従う期間
* **ログ類**: セキュリティ・運用のため、通常は一定期間でローテーションまたは削除します（期間は運用方針により変更される場合があります）。

## 第4条（第三者提供）

開発者は、以下の場合を除き、お客様の個人情報を第三者に提供することはありません。

* **お客様の同意がある場合**
* **法令に基づく場合**
* **生命・身体・財産の保護のために必要な場合**
* **公衆衛生の向上・児童の健全な育成のために特に必要な場合**
* **国の機関・地方公共団体・その委託先による法令の定める事務の遂行に協力する必要がある場合**

なお、クラウド事業者やAI/OCR基盤への **委託・処理** は、サービス提供に必要な範囲で行われます。これは「第三者への売買目的の提供」とは異なりますが、当該事業者が定めるプライバシーポリシーが適用される場合があります（次条参照）。

## 第5条（外部サービスとの連携）

本アプリは、以下の外部サービスと連携しています。

### 5.1 AWS（Amazon Web Services）

* **目的**: アプリケーションのホスティング、データ保存、生成 AI の呼び出し、運用ログ等
* **処理され得る情報**: 端末識別子、入力テキスト、画像（OCR 用）、生成結果、設定、利用ログ等
* **プライバシーポリシー**: [AWS プライバシーポリシー](https://aws.amazon.com/jp/privacy/)

### 5.2 Google Cloud Vision API（OCR）

* **目的**: 画像からのテキスト抽出
* **処理され得る情報**: お客様がアップロードした画像
* **プライバシーポリシー**: [Google プライバシーポリシー](https://policies.google.com/privacy?hl=ja)

### 5.3 Amazon Bedrock

* **目的**: 返信候補の生成、リスク分析、関連する自然言語処理
* **処理され得る情報**: プロンプトとして送信されるテキスト（受信メッセージ、補足事情、返信候補、設定に基づく文脈等）、端末識別子等のメタデータ
* **プライバシーポリシー**: 利用モデル・リージョンに応じ、AWS およびモデル提供者の方針が適用されます。詳細は [AWS の Bedrock に関するドキュメント](https://aws.amazon.com/jp/bedrock/) 等をご参照ください。

### 5.4 Apple Inc.

* **目的**: アプリ配信、App Store を通じた課金・サブスクリプション管理、取引検証に関連する通知等
* **収集・処理**: Apple のプライバシーポリシーおよび App Store の利用規約に従います
* **プライバシーポリシー**: [Apple のプライバシー](https://www.apple.com/jp/privacy/)

### 5.5 サービス別データ送信の明細

本アプリが外部サービスに送信する主なデータは以下のとおりです。

1. **Amazon Bedrock（返信候補生成・リスク分析）**
   * 受信メッセージ本文
   * 補足事情テキスト（任意入力）
   * 返信候補の再分析時に指定した返信テキスト
   * 関係・目的・トーン・長さ等の設定情報
   * 端末識別子、アプリバージョン等の最小限メタデータ

2. **Google Cloud Vision API（OCR）**
   * OCR対象としてお客様が選択した画像データ
   * OCR実行に必要な最小限のリクエスト情報

## 第6条（お客様の権利）

お客様は、個人情報保護法その他の法令に従い、当該法令で認められる範囲で、保有個人データの開示、訂正、利用停止、消去等を請求できる場合があります。請求はお問い合わせ窓口（第10条）から行ってください。法令により対応できない場合があります。

## 第7条（データの削除）

### 7.1 アプリ内での削除

* 履歴の削除機能が提供されている場合、お客様は当該機能により履歴を削除できます。
* 削除処理が非同期である場合、反映まで時間がかかることがあります。
* 削除後の復元は原則としてできません。

### 7.2 アプリのアンインストール

* アンインストールにより端末内のデータが削除される場合がありますが、サーバー側に残存するデータは別途削除が必要な場合があります。詳細はアプリ内の説明およびお問い合わせに従ってください。

## 第8条（未成年者の個人情報）

未成年の方が本サービスを利用される場合は、保護者の方の同意と指導のもとでご利用ください。

## 第9条（プライバシーポリシーの変更）

1. 開発者は、必要に応じて本ポリシーを改定することがあります。
2. 変更内容は、アプリ内表示、本ポリシー掲載ページの更新、その他開発者が適当と判断する方法で告知します。告知後に利用を継続された場合、改定後の内容に同意したものとみなすことがあります。
3. 法令上お客様の同意が必要な変更については、法令に従います。
